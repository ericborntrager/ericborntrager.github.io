Programming Contributions

  • Created an interactions system
    • Allows player to interact with objects and NPCs
    • Abstract and able to be implemented for specific use cases. For example:
      • Opening an in-game computer
      • Using an in-game door
      • Starting dialogue with NPCs
  • Implemented an easily expandable system for the integration of music and SFX
    • Created dictionaries to correspond a key to the directory path of a given sound
      • Created one for music and another for SFX
      • An enum was used to create easily referencable key values for the music and SFX dictionaries
    • Created a music event manager script
      • Listened for scene changes and changed music according to the "default track" of a given scene
        • Default tracks were found by looking them up in a dictionary that corresponded scene name to a key value (enum type)
